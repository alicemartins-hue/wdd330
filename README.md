# Movie Picker

Movie Picker is a web application developed with HTML, CSS and vanilla JavaScript.

Users can:

- Search movies
- Browse popular movies
- Filter movies by genre
- Pick a random movie
- View movie details in a modal window
- Save movies to a watchlist
- Remove movies from the watchlist

## APIs Used

- TMDb API
- OMDb API

## Technologies

- HTML
- CSS
- JavaScript (ES Modules)
- LocalStorage
